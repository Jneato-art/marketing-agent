# Brands

One folder per brand the agent works on. The agent creates `brands/<brand>/` automatically during intake and fills:

- `brief.md` — from the 5 intake questions (template: `templates/brand-brief.md`)
- `research.md` — from the market-research + competitor-analysis skills
- `assets.md` — links to generated creative, copy, and live campaigns

Keep brand folders self-contained so the agent can load everything it knows about one brand at once.
