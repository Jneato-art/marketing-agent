# Experiments Log

_Append a dated entry after every test. Newest at the top. This is how the agent learns._

## Format
```
## <date> | <brand> | <experiment name>
- Tried: <what, on what budget>
- Result: <numbers vs baseline>
- Learning: <the durable takeaway>
- Next: <the follow-up test>
```

---

## Example (delete once you have real entries)

## 2026-06-16 | Example brand | TikTok hook test
- Tried: 3 hooks (taste, convenience, price) at $50 each.
- Result: "convenience" 2.1% CTR vs 0.6% baseline; others flat.
- Learning: lead with convenience for this audience.
- Next: scale convenience angle to $200; test 2 new variations.
