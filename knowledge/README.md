# Knowledge — the agent's memory

This folder is what makes the agent self-improving. The agent **reads** it before every task and **writes** to it after, so each run starts smarter than the last.

```
index.md                  Table of contents + durable benchmarks (read first)
brands/<brand>/
  brief.md                Positioning, audience, voice, constraints
  research.md             Market + competitor findings, with sources
  assets.md               Links to created creative, copy, campaigns
experiments/
  experiments-log.md      Dated log of what was tried and what happened
playbooks-learned/
  <name>.md               Tactics that won, promoted to reusable practice
```

See [../docs/self-learning.md](../docs/self-learning.md) for the full protocol. Keep entries short, dated, and specific.
