# Learned Playbooks

When a tactic beats its benchmark **twice**, the analyst promotes it here as a reusable, brand-specific playbook. These take priority over the generic `playbooks/` because they are proven for *your* brands.

Naming: `<brand>-<tactic>.md`, e.g. `coldbrew-convenience-ugc.md`.

Each file should capture: the situation, the exact tactic, the numbers that proved it, and how to repeat it.
