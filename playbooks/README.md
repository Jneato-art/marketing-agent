# Playbooks

Proven, situation-specific sequences. The agent picks the playbook that matches the goal, then adapts it to the brand's research — it never copies blindly.

| Playbook | Use when the goal is… |
|---|---|
| `product-launch` | A concentrated burst of attention for a new product/offer |
| `growth-loops` | Compounding growth after early traction |
| `seo-content` | Durable inbound traffic that builds an asset |
| `paid-acquisition` | Fast, testable, controllable growth |
| `organic-social` | An owned audience and brand affinity |

Each playbook ends with a "Next 3 actions" template so the agent always leaves the user with something to do today.
