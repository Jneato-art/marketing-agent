# Playbook: Growth Loops

Use when you have product-market fit signals and want compounding, not one-off, growth. A loop is a cycle where the output of one user's action brings in the next user.

## Loop types
- **Referral loop**: happy customer -> refers friend -> friend buys -> refers again. Add an incentive both sides want.
- **Content loop**: user creates/shares -> content gets seen -> new users arrive -> create more.
- **Paid loop**: spend -> acquire customer -> LTV > CAC -> reinvest profit into more spend.
- **Network loop**: each new user makes the product more valuable to others.

## How to build one
1. Pick ONE loop that fits the product and audience.
2. Map every step and the drop-off at each.
3. Find the weakest step and fix it first (the loop is only as fast as its slowest gate).
4. Add the right incentive/trigger to push users to the next step.
5. Measure loop speed (cycle time) and amplification (how many new users per existing user).

## The math that matters
For a referral loop, the viral coefficient k = (invites sent per user) x (conversion rate). k > 1 = exponential. Even k = 0.5 meaningfully cuts CAC.

## KPIs
Cycle time, viral coefficient, LTV:CAC ratio (aim 3:1+).

## Next 3 actions template
1. Choose the single loop to build.
2. Instrument each step to find the drop-off.
3. Ship one incentive to lift the weakest step.
